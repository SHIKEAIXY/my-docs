## 关于依赖


::: info 依赖问题

脆脆鲨插件中依赖均可通过此命令下载，正常情况下无需下载

::: code-tabs

@tab pnpm

```bash
pnpm install -P
```

@tab npm

```bash
npm install
```

@tab:active yarn

```bash
yarn install
```
:::
